Automatic Registration

Manak offers auto case and suite registation with a user friendly API. Auto registration can be done anywhere where manak.hpp is included. It cannot be inside any class or a function but it can be inside a namespace. For support of timing macros inside auto registration see All About Timing.

Auto Registration With Simple Module

Simplest auto benchmark case registration -

#include <iostream>
#define MANAK_SIMPLE_BENCHMARK_MODULE bench
#define MANAK_AUTO_BENCHMARK_MAIN
#include <manak/manak.hpp>
MANAK_AUTO_BENCHMARK_CASE(B1)
{
Code;
}

Any auto registration registers the case or suite to the current benchmark suite. When no benchmark suite is set the current benchmark suite is set to master suite. So in the example above 'MANAK_AUTO_BENCHMARK_CASE' will register 'B1' to master suite.

When iteration or tolerance is not specified for any benchmark case the default will be used. See MANAK_DEFAULT_ITERATIONS and MANAK_DEFAULT_TOLERANCE for more details.

See Comparison Framework for Simple Module to understand the comparison framework in simple module.

MANAK_AUTO_BENCHMARK_CASE

Creates benchmark case with name, and registers it to the current benchmark suite. Iterations and tolerance for the benchmark case are taken to be defaults.

MANAK_AUTO_BENCHMARK_CASE_I

Creates benchmark case with name and iterations, and registers it to the current benchmark suite.

MANAK_AUTO_BENCHMARK_CASE_I(B1, 10)
{
Code;
}

MANAK_AUTO_BENCHMARK_CASE_T

Creates benchmark case with name and tolerance, and registers it to the current benchmark suite.

MANAK_AUTO_BENCHMARK_CASE_T(B1, 50)
{
Code;
}

MANAK_AUTO_BENCHMARK_CASE_IT

Creates benchmark case with name, iterations and tolerance, and registers it to the current benchmark suite.

MANAK_AUTO_BENCHMARK_CASE_IT(B1, 10, 50)
{
Code;
}

MANAK_ADD_BENCHMARK

'MANAK_ADD_BENCHMARK' can be used to register manually generated benchmark case. This is especially useful when written function needs to be added as benchmark case.

#include <iostream>
#define MANAK_SIMPLE_BENCHMARK_MODULE bench
#define MANAK_AUTO_BENCHMARK_MAIN
#include <manak/manak.hpp>
int fun()
{
Code to benchmark;
}
MANAK_ADD_BENCHMARK(MANAK_BENCHMARK_CASE(B1, fun));

'MANAK_BENCHMARK_CASE' is a directive for manual registration. For more such directives see Manual Registration.

The function to be registered must have no parameters. The function with parameters can used to generate parametrized benchmark cases. See Complete guide on Parametrized Benchmark Cases for more information. Also see Advance Function Registration for registration of complex functions such as overloaded functions, template functions and class member functions.

MANAK_AUTO_BENCHMARK_SUITE

Creates a new benchmarking suite with given name and registers it to the current benchmarking suite. It also sets the newly generated benchmarking suite as the current suite.

#include <iostream>
#define MANAK_SIMPLE_BENCHMARK_MODULE bench
#define MANAK_AUTO_BENCHMARK_MAIN
#include <manak/manak.hpp>
MANAK_AUTO_BENCHMARK_SUITE(Suite1);
MANAK_AUTO_BENCHMARK_CASE(B1)
{
Code;
}

In the above example 'Suite1' will be registered under master suite and 'B1' will be registered under 'Suite1'.

MANAK_AUTO_BENCHMARK_END

Sets the current benchmark suite to the parent of current of benchmark suite.

#include <iostream>
#define MANAK_SIMPLE_BENCHMARK_MODULE bench
#define MANAK_AUTO_BENCHMARK_MAIN
#include <manak/manak.hpp>
MANAK_AUTO_BENCHMARK_SUITE(Suite1);
MANAK_AUTO_BENCHMARK_CASE(B1)
{
Code;
}
MANAK_AUTO_BENCHMARK_SUITE_END();
MANAK_AUTO_BENCHMARK_CASE(B2)
{
Code;
}

In the above example 'B1' will be registered under 'Suite1' but 'B2' will be registered under master suite.

Parametrized Benchmark cases

Parametrized benchmark cases can be registered with macro 'MANAK_ADD_BENCHMARK'.

#include <iostream>
#define MANAK_SIMPLE_BENCHMARK_MODULE bench
#define MANAK_AUTO_BENCHMARK_MAIN
#include <manak/manak.hpp>
int fun(size_t a)
{
for(size_t i = 0;i < a;i++);
}
MANAK_ADD_BENCHMARK(MANAK_CREATE_BENCHMARK_WITH_TEMPLATE(ForLoops, fun)->AddArgs((size_t)1000)->AddArgs((size_t)2000));

For more advance parametrized benchmark options check Complete guide on Parametrized Benchmark Cases.

For example on simple module auto registration check Example.

Auto Registration With Normal Module

Simplest auto benchmark case registration -

#include <iostream>
#define MANAK_BENCHMARK_MODULE bench
#define MANAK_AUTO_BENCHMARK_MAIN
#include <manak/manak.hpp>
MANAK_AUTO_BENCHMARK_CASE(B1, lib1)
{
Code;
}
